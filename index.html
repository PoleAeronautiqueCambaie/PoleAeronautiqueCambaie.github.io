Règles de confidentialité

Cambaie Aero Info est éditée par le Pôle Aéronautique de Cambaie, Association de la loi du 1er juillet 1901, 58 Rue Henri Cornu, St Paul 97460, La Réunion..
Le terme Application s’applique à l’application pour mobile Android ou l'application web nommée "Cambaie Aero Info".
Le terme Serveur Metar s’applique au composant logiciel "metarserve" avec lequel communique l’Application. Nos règles de confidentialité expliquent quelles données nous collectons depuis l’Application et les raisons de cette collecte, comment nous utilisons / traitons ces informations et à quels tiers nous pourrions éventuellement les divulguer.

Données personnelles

Depuis notre Application, nous ne collectons aucune donnée personnelle (par exemple: les noms, les adresses, les numéros de téléphone, les adresses de courrier électronique ou les adresses IP complètes).

Données relatives à la communication

Lorsque vous utilisez notre Application via les services de télécommunications pour communiquer avec un Serveur Metar, des données relatives à la communication (par exemple, l’adresse IP) sont automatiquement générées. Le Serveur Metar ne conserve pas ces données, elles sont utilisées uniquement pour l’établissement d’une communication entre l’Application et un Serveur Metar.

Permissions demandées par l’Application

Utilisation de la data internet

Nous utilisons uniquement une connexion internet pour communiquer avec le Serveur Metar.

Données personnelles non collectées automatiquement

Nous pouvons collecter uniquement des informations en cas de panne de votre mobile lors de l’utilisation de l’Application. Les informations suivantes peuvent être collectées :

  • Le type d’appareil utilisé,
  • La version d’Android,
  • L’opérateur mobile,
  • Le type de connexion active (WiFi ou réseau mobile),
  • La stacktrace décrivant dans quelle section du code le crash s’est produit.

Nous ne collectons aucune information personnelle identifiable . Nous ne vendons pas, n’échangeons ni transférons ces informations à des tiers. Cependant, le Pôle Aéronautique de Cambaie s’appuie sur des logiciels fournis par d’autres éditeurs ; leurs règles de confidentialité sont aussi applicables.
